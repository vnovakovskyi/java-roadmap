# Books & Resources

A curated, **free-first** list. You do **not** need to read all of this — each topic note links the
specific resource you need at that moment. This page is the big-picture reference.

> [!info] How to use this
> Pick **one** primary book to read alongside the path, lean on the free official docs for detail,
> and use videos when a concept won't click from text. Tiers: **must** = core, **good** =
> recommended, **optional** = when you want more. Back to home: [[Java Developer Roadmap - MOC]].

## Books

### For the absolute beginner (pick one as your main book)
- **Head First Java, 3rd ed.** — Sierra, Bates & Gee. *(must)* The friendliest on-ramp; visual,
  hands-on, assumes nothing. Covers Java 17+. The single best companion for Modules 1–5.
- **Learn Java the Easy Way** — Bryson Payne, or **Java: A Beginner's Guide** — Herbert Schildt.
  *(optional)* Alternatives if Head First's style isn't for you.

### Level-up (read after you can write basic classes)
- **Effective Java, 3rd ed.** — Joshua Bloch. *(good)* The book that teaches you to write *good*
  Java. Don't read it first — revisit its items as you finish Modules 3–5 and 7.
- **Modern Java in Action** — Urma, Fusco & Mycroft. *(good)* Best deep treatment of lambdas,
  streams, and modern features (pairs with Module 5).

### Reference / later
- **Java Concurrency in Practice** — Goetz et al. *(optional)* The classic on threads; beyond junior
  scope, but the reference when you get there.
- **Spring in Action, 6th ed.** — Craig Walls. *(good)* A thorough Spring Boot companion for Module 9.
- **Spring Start Here** — Laurențiu Spilcă. *(good)* Gentler Spring intro if Spring in Action feels heavy.

## Free official documentation (your default detail source)
- **[Dev.java](https://dev.java/learn/)** — Oracle's official, modern Java tutorials. *(must)*
- **[The Java Tutorials](https://docs.oracle.com/javase/tutorial/)** — older but comprehensive. *(good)*
- **[Java 21 API docs (Javadoc)](https://docs.oracle.com/en/java/javase/21/docs/api/)** — look up any
  class/method. *(must — learn to read these early)*
- **[Spring Boot reference + Guides](https://spring.io/guides)** — short, official how-tos. *(must for M9)*
- **[Baeldung](https://www.baeldung.com)** — huge library of practical Java/Spring articles. *(good)*

## Free courses & video
- **[freeCodeCamp — Java Full Course](https://www.youtube.com/c/Freecodecamp)** (YouTube) *(good)*
- **[Amigoscode — Java/Spring](https://www.youtube.com/@amigoscode)** *(good)* — clear Spring Boot intros.
- **[Java Brains](https://www.youtube.com/@Java.Brains)** *(good)* — Spring & Java concept explainers.
- **[University of Helsinki — Java Programming MOOC](https://java-programming.mooc.fi)** *(must-try)* —
  free, full, exercise-driven course; an excellent structured alternative for Modules 1–4.

## Practice & community
- **[Exercism — Java track](https://exercism.org/tracks/java)** *(must)* — guided exercises with
  mentor feedback. Great for Modules 2–5.
- **[Codingbat — Java](https://codingbat.com/java)** *(good)* — tiny problems to drill syntax.
- **[r/learnjava](https://www.reddit.com/r/learnjava/)** and its wiki *(good)* — ask questions, read
  the FAQ.
- **[Stack Overflow](https://stackoverflow.com)** — search first; you'll live here.

## Tooling references
- **[SDKMAN!](https://sdkman.io)**, **[Adoptium/Temurin](https://adoptium.net)** — installing Java.
- **[Maven](https://maven.apache.org/guides/)** and **[Gradle](https://docs.gradle.org)** user guides.
- **[Pro Git book](https://git-scm.com/book/en/v2)** *(good, free)* — the definitive Git reference.
