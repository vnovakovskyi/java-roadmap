# Loops (for and while)

> [!info] Topic
> **Area:** Core Java · **Module:** 2 · **Priority:** must · **Est:** 4h
> **Why this matters:** Loops let you repeat work — iterate over data, retry, accumulate. They're
> half of all the algorithms you'll ever write.

## Learning Objectives
- Write `for`, `while`, and `do-while` loops.
- Use the enhanced for-each loop over collections/arrays.
- Control loops with `break` and `continue` and avoid infinite loops.

## Learn
- [ ] Read: [dev.java — "Loops / iterating"](https://dev.java/learn/language-basics/).
- [ ] Read about the for-each loop.

## Concepts
- [ ] `for (init; condition; update)` — best when you know the count or need an index.
- [ ] `while (condition)` — repeat until a condition changes; `do-while` runs at least once.
- [ ] **For-each**: `for (var item : items)` — cleanest for iterating a collection/array.
- [ ] `break` exits the loop; `continue` skips to the next iteration.
- [ ] Off-by-one errors and infinite loops are the classic bugs — check your bounds and updates.

## Practice
**Goal:** Drill iteration patterns.
**Steps:**
- [ ] Print the numbers 1–10 with a `for` loop; then only the even ones with `continue`.
- [ ] Sum the numbers 1–100 in a loop and print the total.
- [ ] Read numbers from the user until they type `0` (a `while` loop), then print the count.
